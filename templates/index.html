Secure File Transfer

Instructions

The user gets to choose from one of the following options:

1. He gets to upload the file to the cloud in a secured manner
2. He gets to choose from thelist of all the different files that are present on the cloud provided he has the required credentials for decrypting the file.
3. He can also download public key associated with the user with whom the file transfer is to take place.
4. To upload his file to cloud storage the user needs to be registered so that his file can be shared with the other desired person in an instant.
